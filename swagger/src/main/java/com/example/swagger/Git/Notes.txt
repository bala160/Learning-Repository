for first time take Git clone <URL REPO> in you creadted dir path

Eg :  git clone https://github.com/bala160/Learning-Repository.git

step 1 : pull

Eg : PS C:\Users\Balakrishnan\OneDrive\Desktop\GitHub\Learning-Repository\swagger\src\main\java\com\example\swagger> git pull

Step 2 : add

Eg : PS C:\Users\Balakrishnan\OneDrive\Desktop\GitHub\Learning-Repository\swagger\src\main\java\com\example\swagger> git add .\Git

step 3 : commit

Eg : PS C:\Users\Balakrishnan\OneDrive\Desktop\GitHub\Learning-Repository\swagger\src\main\java\com\example\swagger> git commit -m  "Modified variable name 2"
[master 2bcbf21] Modified variable name 2
 1 file changed, 2 insertions(+), 2 deletions(-)

step 4 : push (merge conflict remove changes)

Eg : PS C:\Users\Balakrishnan\OneDrive\Desktop\GitHub\Learning-Repository\swagger\src\main\java\com\example\swagger> git push

Step 5 : commit all changes file

Eg : PS C:\Users\Balakrishnan\OneDrive\Desktop\GitHub\Learning-Repository\swagger\src\main\java\com\example\swagger> git commit -am "Changes done in multiple file"
     [master 3ab45b5] Changes done in multiple file
      2 files changed, 4 insertions(+), 2 deletions(-)


Step 6: To create new branch

Eg : PS C:\Users\Balakrishnan\OneDrive\Desktop\GitHub\Learning-Repository\swagger\src\main\java\com\example\swagger> git checkout -b "UAT"
     Switched to a new branch 'UAT'

Step 7 : To find branch (current branch having * symbol)

Eg : PS C:\Users\Balakrishnan\OneDrive\Desktop\GitHub\Learning-Repository\swagger\src\main\java\com\example\swagger> git branch
     * UAT
       master

Step 8 : To checkout another branch

Eg : PS C:\Users\Balakrishnan\OneDrive\Desktop\GitHub\Learning-Repository\swagger\src\main\java\com\example\swagger> git checkout SIT
     Switched to branch 'SIT'

step 9 : To Merge another branch

Eg : PS C:\Users\Balakrishnan\OneDrive\Desktop\GitHub\Learning-Repository\swagger\src\main\java\com\example\swagger> git merge master
     Updating 29089b3..4a69940
     Fast-forward
      swagger/src/main/java/com/example/swagger/Git/Notes.txt | 8 +-------
      1 file changed, 1 insertion(+), 7 deletions(-)
